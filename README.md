# NikkiScript
无限暖暖电脑版本的小脚本
# 功能说明

\========================================
* 按 E 键：切换自动按 F 键
* 按 R 键：切换自动蓄力攻击
* 鼠标中键：按住自动收杆，释放后按 F
\========================================
* 按 Ctrl+ ` ：暂停/恢复脚本
* 按 Ctrl+H：隐藏/显示脚本面板
* 按 Ctrl+Q：退出脚本
* 按住 Ctrl + 拖动：移动状态窗口
\========================================
        未来可能失效：
* 按 [ 键：滞空时切换飞鸟套的远距离滑翔（快速按 W）
\========================================

检测游戏进程：{X6Game-Win64-Shipping.exe}

注意：仅当游戏全屏运行时功能才会激活,在其他页面不激活

\========================================


2025年5月25日更新：
新增：
按 X 键：开启拽鱼模式(切换按住AD，点中键关闭)
鼠标中键：按住自动收杆(同时关闭拽鱼模式)，释放后按 F
